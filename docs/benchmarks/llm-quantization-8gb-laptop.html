How it was measured

Fixed prompts, temperature 0, answers graded by deterministic code (a number, a letter, JSON asserts) — never an LLM judge. Hardware and exact model tags are recorded inside each data file. Full protocol: methodology.

Questions this answers

Which quantization fits an 8GB laptop?

Only Q4 (~4.8GB). FP16 (~16GB) won't load; Q8 (~8.5GB) swap-thrashed to ~0.03 tok/s.

How much accuracy does Q4 keep?

87% on the 15-task graded exam in our run.
