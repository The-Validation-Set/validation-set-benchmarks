Is local AI actually cheaper than the cloud?

Local AI vs cloud API: what one always-on agent really costs

A managed API ≈ $7,200/yr per always-on agent; an owned rig breaks even ~month 5.

Where is the break-even between renting and owning?

Self-hosting an LLM: the break-even, measured on both sides

Same model both sides: cloud ~4.3× faster, but the owned rig pays for itself ~month 5 and banks ~$4,500 in year one.

How many agents before a cheap rig chokes?

How many AI agents can an 8GB laptop actually run?

Usable knee = 2 agents (Ollama serialises; tail latency ~6s → 60s+). The 8B won't even load. ≈ $1,326/agent/yr vs $7,200 rented.

Is a cheap local model actually smart enough?

3B vs 8B vs 14B on one 8GB laptop — the same graded exam

3B: 80% at ~26 tok/s. 8B: 87% at ~6 tok/s — one extra answer for 4× the wait. 14B: never finished. The gap is all multi-step math.

What does the newest frontier model actually cost?

Claude Fable 5 pricing: exactly 2× Opus 4.8

$10 in / $50 out per 1M tokens — exactly 2× Opus 4.8. Whether that matters depends on cost per correct answer, not per token.

How much better is the frontier, per dollar?

A frontier model took the local exam: 15/15 — at 657× the cost per answer

Claude Fable 5: 15/15, whole exam ≈ $0.02 — but ≈657× the free local 3B's cost per correct answer. Everything it fixed was math.

Does self-hosting ever beat a hosted API?

Self-hosting an 8B vs a hosted API: there is no break-even

Same 8B, laptop vs hosted: hosted ~56× faster AND cheaper per token → no break-even at any volume for this workload.

Do you even need a vector database?

Vector DB vs 10 lines of NumPy: the silent recall collapse

Chroma (defaults) slid to ~49.5% recall@10 by 200k vectors — no error, no warning. Brute-force NumPy held 100% at ~23ms.

Is one local runtime secretly faster?

Ollama vs MLX on Apple Silicon: a measured dead heat

Same 3B, same base M3: Ollama 23.95 vs MLX 23.8 tok/s — 0% apart, within noise. The real lever is local vs cloud, not runtime.

How many injections get through an undefended agent?

Indirect prompt injection on a local model: 80% obeyed the attacker

Undefended: 80% of textbook OWASP LLM01 payloads obeyed. One defense (context segregation): 0% — and 0% still ≠ solved.

Which quantization actually fits consumer hardware?

FP16 vs Q8 vs Q4 on an 8GB laptop: only one of them loads

FP16 (~16GB) and Q8 (~8.5GB) don't fit — Q8 swap-thrashed to ~0.03 tok/s. Q4 (~4.8GB) runs and scored 87% on the exam.

Does the 'smarter' 4-bit beat the one you're running?

Gemma 3 12B: training-aware QAT vs naive PTQ, graded

A flat 80% / 80% tie on accuracy. QAT buys ~9% speed (32 vs 29 tok/s) for ~0.8GB more — not accuracy. Grab it where it's shipped.

Keep it local, or pay the cloud?

Route only your measured weakness to the cloud: −73% bill, same score

Local 3B is only weak on math. Escalating just that bucket to a frontier model: 100% accuracy at 27% cloud calls ≈ 73% off the modeled bill.

Does a re-ranker actually help your RAG?

RAG re-rankers fix order, not recall — measured

The bi-encoder already nailed rank-1 (easy 92→100; hard 100→100 — re-ranker added 0) at ~38ms/query. Fix recall first; order later.
