Methodology

The rules every number on this site obeys — and the labels that keep measured and modeled honest.

Measured means measured

Accuracy and throughput come from real runs on named hardware with exact model tags, recorded inside each data file. Runs use fixed prompts and temperature 0 so anyone can reproduce them bit-for-bit where the backend allows it.

Graded by code, never by vibes

The 15-task exam covers arithmetic, logic, code, format-following, and extraction. Every answer is checked by a deterministic grader — an exact number, a letter, JSON asserts. No LLM-as-judge anywhere in the pipeline, because a judge model smuggles its own biases into the score.

Modeled means labelled

Dollar figures (electricity, cloud pricing, cost per correct answer) are models built from labelled public inputs — utility tariffs, provider price pages — cited inside the data files. They are marked modeled wherever they appear, so you always know which kind of number you're reading.

Scope is part of the result

A result on a 3B dense model on an 8GB laptop is claimed for that regime, not for the frontier. Where a finding stopped generalising in our own testing, the page says so.

Zero sponsorship

No vendor pays for placement or verdicts. When that ever changes for a specific piece of content, it will be disclosed on the content itself.

Break it, get credited

Every harness is public. Run it on your hardware; if your numbers corroborate or contradict ours, open an issue — you get a row in CONTRIBUTORS.md. Contradictions are the most valuable thing you can send.
